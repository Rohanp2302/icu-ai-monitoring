# ✅ FRESH START GUIDE - CLEAR CACHE & RUN

## IF YOU'RE SEEING THE OLD DASHBOARD

The new patient input page might be cached in your browser. **DO THIS:**

---

## 🔧 STEP 1: Hard Refresh Browser

Press one of these keyboard shortcuts:
- **Chrome/Firefox/Edge**: `Ctrl + Shift + Delete` (Opens cache clear dialog)
- **All browsers**: `Ctrl + Shift + R` (Hard refresh)
- **Alternative Mac**: `Cmd + Shift + R`

Then:
1. Close the browser completely
2. Restart the Flask server (see below)
3. Open fresh browser window

---

## 🚀 STEP 2: Fresh Start the Server

**Option A - Clean Kill & Restart:**
```bash
# Kill any running Flask process
taskkill /F /IM python.exe

# Wait 2 seconds, then:
E:\ANACONDA\envs\icu_project\python.exe e:/icu_project/app.py
```

**Option B - Start Fresh:**
1. Close terminal/PowerShell completely
2. Open NEW PowerShell or Command Prompt
3. Run:
```powershell
cd E:\icu_project
E:\ANACONDA\envs\icu_project\python.exe app.py
```

---

## 🌐 STEP 3: Open in Fresh Browser

1. Close all browser tabs/windows
2. Open a NEW browser window

## 🐛 IF STILL NOT WORKING

### Check 1: Browser Developer Console
Press `F12` to open DevTools → Check Console tab for errors

### Check 2: Server Terminal
Look for errors in the terminal where Flask is running

### Check 3: Verify Server is Running
Go to: http://localhost:5000/api/health
Should show:
```json
{
  "status": "ok",
  "model_loaded": false,
  "model_info": {...}
}
```

### Check 4: Try Different URL
- If stuck on old page, try: http://localhost:5000/?t=nocache
- Or clear ALL browser data: Settings → Clear browsing data → All time
