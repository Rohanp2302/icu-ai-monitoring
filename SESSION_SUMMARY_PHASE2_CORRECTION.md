## Key Changes to Pipeline

### Training Pipeline (Before → After)

```
BEFORE:
───────
X, y = load_data()
X_train, X_test, y_train, y_test = train_test_split(X, y)
model.fit(X_train, y_train)                    # Raw data, no scaling


AFTER:
──────
X, y = load_data()
X_train, X_test, y_train, y_test = train_test_split(X, y)
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train) # Fit on train only
X_test_scaled = scaler.transform(X_test)      # Apply using train stats
model.fit(X_train_scaled, y_train)             # Normalized data
```

### Evaluation Pipeline (Before → After)

```
BEFORE:
───────
model = load_model()
X_test_scaled = scaler.fit_transform(X_test)  # Fit on test (leak!)
predictions = model(X_test_scaled)              # Mismatched input scale


AFTER:
──────
checkpoint = load_checkpoint()                  # Has scaler mean/scale
scaler.mean_ = checkpoint['scaler_mean']
scaler.scale_ = checkpoint['scaler_scale']
X_test_scaled = scaler.transform(X_test)        # Use train statistics
predictions = model(X_test_scaled)              # Consistent input scale
```

## What We Learned (Lessons for Future Phases)

### ✓ Best Practices Established

1. **Data Pipeline Order**
   - SPLIT first (before preprocessing)
   - FIT scaler on training set only
   - APPLY scaler using training statistics to all sets

2. **Preprocessing Isolation**
   - StandardScaler never fit on full dataset
   - Cross-validation: fresh scaler per fold
   - Prevents information leakage from val/test → train

3. **Validation Framework**
   - Always include baseline models
   - Report train/val/test AUC gap
   - Flag any CV results with 0% or 100% performance
   - Verify stability across folds (variance < 0.05)

4. **Suspicious Metrics Investigation**
   - >99% AUC on small dataset? Investigate!
   - CV fold variance 0-100%? Red flag!
   - Train-test gap >20%? Likely overfitting/leakage

### ✓ Debugging Checklist

Created for use in Phase 3:
- [ ] Is preprocessing done correctly? (Split → Normalize)
- [ ] Is scaler fit on training data only?
- [ ] Are val/test preprocessed using train statistics?
- [ ] Does model generalize? (train/test gap <15%)
- [ ] Are results stable across CV folds?
- [ ] Do baselines make sense contextually?
