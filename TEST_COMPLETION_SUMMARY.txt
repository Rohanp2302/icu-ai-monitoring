## HOW TO USE IT NOW

### Option 1: Web Dashboard
Visit: http://localhost:5000
Upload a CSV file with patient data
See predictions instantly

### Option 2: API Testing
```bash
# Batch upload
curl -F "file=@test_patients.csv" http://localhost:5000/api/predict

# Single patient
curl -X POST http://localhost:5000/api/predict \
  -d 'data={"patient_id":"P1","HR_mean":75,"RR_mean":18,"SaO2_mean":97,"age":45}'

# Health check
curl http://localhost:5000/api/health
```

### Option 3: Python Integration
```python
import requests, json
patient = {'patient_id': 'P1', 'HR_mean': 110, 'RR_mean': 26, 'SaO2_mean': 89, 'age': 74}
r = requests.post('http://localhost:5000/api/predict',
                  data={'data': json.dumps(patient)})
print(r.json()['predictions'][0]['mortality_percent'])  # "9.1%"
```
