## 🚀 QUICK START - Choose Your Method

### METHOD 1: Double-Click Batch File (EASIEST)
1. Open File Explorer
2. Navigate to: `E:\icu_project`

5. Open browser: `http://localhost:5000/`

1. Open PowerShell
2. Run:
```powershell

```
3. Open browser: `http://localhost:5000/`

## 🛑 IF YOU GET ERRORS

### "Port 5000 in use"
The port is already being used. Either:
- Kill existing process using port 5000
- Edit `app.py` line: `app.run(..., port=5001)`

### "Module not found"
Ensure you're using the correct Python:
```powershell
E:\ANACONDA\envs\icu_project\python.exe app.py
```

### "Permission denied"
For PowerShell in windows:
```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
```

### "Connection refused"
- Make sure app is still running in terminal
- App might have crashed - check terminal for errors
