## 🛠️ TROUBLESHOOTING

### Problem: Still seeing old page
**Solution:**
1. `Ctrl + Shift + Delete` → Clear all
2. Close browser completely
3. Restart Flask
4. Open new browser window

### Problem: "Port 5000 in use"
**Solution:**
```bash
taskkill /F /IM python.exe
# Wait 3 seconds
python app.py
```

### Problem: "Module not found"
**Solution:**
```bash
E:\ANACONDA\envs\icu_project\python.exe app.py
```
(Use full Python path)

### Problem: Cannot connect
**Solution:**
- Check Flask is running (terminal should show "Running on...")
- Try: `http://localhost:5000/api/health`

## 🎓 SYSTEM ARCHITECTURE

```
Browser
  ↓
http://localhost:5000/
  ↓
Patient Input Page (patient_upload.html)
  ├─ CSV Upload OR
  └─ Manual Form
     ↓
     API Endpoint
     ├─ /api/upload-csv (CSV data)
     └─ /api/analyze-patient (Form data)
        ↓
        analyze_patient_data()
        ├─ Create temporal data
        ├─ Run prediction
        ├─ Check validation layers
        ├─ Detect medicine interactions
        └─ Generate family explanations
           ↓
           Store in patient_data_store
           ↓
           Redirect to /analysis/<patient_id>
              ↓
              Analysis Page (analysis.html)
              ├─ Fetch data from API
              ├─ Render dashboard
              └─ Display all sections
```
