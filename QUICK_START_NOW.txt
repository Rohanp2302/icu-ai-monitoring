# 🚀 QUICK START GUIDE

## In 3 Steps:

### Step 1: Start the Server
```bash
cd e:\icu_project
python app.py
```

### Step 2: Open the Browser
Navigate to:
```
http://localhost:5000/
```

### Step 3: Input Patient Data

**Option A - CSV Upload:**
1. Click "Upload CSV File"
2. Drag & drop patient CSV (or click to browse)
3. Click "Analyze CSV Data"

**Option B - Manual Form:**
1. Fill patient name, age, gender
2. Enter vital signs (HR, RR, O2, BP, Temp)
3. Add optional info (medications, labs, ICU reason)
4. Click "Analyze Patient Data"

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Patient not found" | Browser cache issue - Clear and refresh |
| "Module not found" | Ensure in correct directory: `cd e:\icu_project` |
| "Port 5000 in use" | Change in code: `app.run(..., port=5001)` |
| Vital signs show "-" | Ensure all required fields entered in form |
| Medications not showing | Check medicine names in database |
